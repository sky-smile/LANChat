# 局域网聊天软件开发计划

## 项目信息
- **项目名称**：LANChat
- **技术栈**：Rust (Axum + Tokio) + Tauri 2.0 + React 18
- **总工期**：12-17周（3-4.25个月）
- **开始日期**：2026-07-15

| 风险 | 影响 | 应对措施 |
